AI Code Detector Pro

Detect AI-generated code in GitHub repositories with advanced analysis

About the Extension

AI Code Detector Pro is a Chrome extension that helps developers, researchers, and security professionals identify whether code in GitHub repositories is AI-generated or human-written. It combines multiple detection methods including fingerprinting, heuristics, stylometry, and GPT-based classification.

Features

  • Analyze GitHub repositories with a single click
  • Detect AI fingerprints from tools like Copilot, Replit, Tabnine, etc.
  • Heuristic & stylometry-based detection
  • Optional GPT/Claude-powered classification for deeper analysis
  • Results dashboard with files, commits, and pull request insights

Privacy

Your data is safe. The extension only analyzes GitHub repository content and does not collect personal data. Optional API keys (OpenAI, Anthropic, GitHub) are stored locally and used only for analysis. See our Privacy Policy for details.

Installation

You can install AI Code Detector Pro directly from the Chrome Web Store .
