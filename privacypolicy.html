Privacy Policy

Effective Date: August 21, 2025

This Privacy Policy explains how the AI Code Detector Pro Chrome extension handles user data. We respect your privacy and are committed to protecting it.

Information We Collect

  • Website Content: The extension only analyzes content from GitHub repositories (source code, commits, pull requests) for the purpose of AI detection.
  • Optional API Keys: If you choose to provide an OpenAI, Anthropic, or GitHub API key, it is stored locally on your device using Chrome's secure storage and never shared with us. Keys are only sent to the respective services you select.

Information We Do Not Collect

The extension does not collect:

  • Personally identifiable information (name, email, address, etc.)
  • Health information
  • Financial or payment data
  • Passwords or authentication credentials (beyond user-provided API keys)
  • Personal communications
  • Location data
  • Web browsing history
  • Keystrokes, mouse movements, or unrelated user activity

Data Usage

- All analysis is initiated manually by the user. - Repository data may be processed locally or sent securely to OpenAI/Anthropic APIs if the user provides their key. - We do not collect, transmit, or sell any user data.

Third-Party Services

When you provide an API key, requests may be sent directly to third-party services such as OpenAI, Anthropic, or GitHub. Please review their respective privacy policies:

  • OpenAI
  • Anthropic
  • GitHub

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with a revised effective date.
